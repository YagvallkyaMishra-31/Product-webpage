# MERN Stack Practice Projects

This repository contains various practice projects and exercises for learning the MERN stack (MongoDB, Express.js, React, Node.js).

## Projects Included

### 1. E-Commerce Website (tempproject)
- Product catalog with pagination
- Search functionality
- Responsive design
- Fetches data from DummyJSON API

### 2. HTTP Basics
- Basic HTTP server implementation
- Understanding request/response cycle

### 3. HTTP Routing
- URL-based routing implementation
- Multiple route handling
- 404 error handling

### 4. Event Emitter
- JavaScript EventEmitter examples
- Event-driven programming concepts

### 5. JavaScript Basics
- Core JavaScript concepts and practice

## Getting Started

1. Clone the repository
2. Navigate to the specific project folder
3. Follow the instructions in each project

## Technologies Used
- Node.js
- JavaScript
- HTML/CSS
- HTTP module
